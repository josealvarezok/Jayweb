## 🚀 English Version

Professional, high-converting landing page built for JAY, a marketing and AI agency. This project features a premium brutalist aesthetic, smooth animations, and a focus on conversion optimization.

### ✨ Features
- **Modern Stack**: React 19 + Vite + Tailwind CSS 4.
- **Premium Design**: Dark mode aesthetic with glassmorphism and custom marquee effects.
- **Conversion Focused**: Integrated Calendly scheduling and Floating WhatsApp contact.
- **Responsive**: Mobile-first approach, fully fluid across all screen sizes.
- **Interactive**: Smooth scroll, hover animations, and dynamic FAQ section.

### 📂 Project Structure
```text
src/
├── components/
│   ├── AboutMe.jsx           # Background and personal story
│   ├── ContactSection.jsx    # Contact info & Calendly integration
│   ├── FAQSection.jsx        # Interactive accordions for common queries
│   ├── FloatingWhatsApp.jsx  # Sticky WhatsApp button
│   ├── Footer.jsx            # Copyright and quick links
│   ├── HeroSection.jsx       # Main impact header with CTA
│   ├── Navigation.jsx        # Sticky navigation menu
│   ├── PartnerBadges.jsx     # Trust signals and certifications
│   ├── ProcessSection.jsx    # Step-by-step working methodology
│   ├── ServicesGrid.jsx      # Detailed service offerings
│   ├── StatsSection.jsx      # Key performance indicators
│   └── SuccessStories.jsx    # Testimonials and case studies
```

## 🚀 Versión en Español

Landing page profesional de alta conversión diseñada para JAY, agencia de marketing e Inteligência Artificial. El proyecto destaca por su estética brutalista premium, animaciones fluidas y enfoque total en resultados.

### ✨ Características
- **Stack Moderno**: React 19 + Vite + Tailwind CSS 4.
- **Diseño Premium**: Estética Dark Mode con glassmorphism y efectos de marquesina personalizados.
- **Enfoque en Conversión**: Integración nativa con Calendly y botón flotante de WhatsApp.
- **Responsive**: Diseño Mobile-first, totalmente fluido en todos los dispositivos.
- **Interactivo**: Scroll suave, animaciones al pasar el mouse y sección dinámica de FAQs.

### 🛠️ Instalación Local

1. Clonar el repositorio: `git clone https://github.com/tu-usuario/JayAgency.git`
2. Instalar dependencias: `npm install`
3. Iniciar servidor de desarrollo: `npm run dev`
